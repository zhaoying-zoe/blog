# blog
blog for zhaoying

## Technology stack
### nodejs 充当后台服务
### express 用来搭建后台 (路由模块化)
### body-parser (express第三方插件,处理post和put请求 传递的参数)
### mongoose 
### Bootstrap@3 前台UI框架
### jquery@3
### swig模板引擎

## Folder description
### views: 模板
### routers: 处理路由
### models: 定义文档模型
### db: 数据库
### public: 静态资源文件夹
